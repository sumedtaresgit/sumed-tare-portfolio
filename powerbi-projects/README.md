**Project 1**: Order Status Check Dashboards.

**Role:** BI Developer.

**Technologies:** Power BI, SQL Server, DAX Studio, Azure Fabric. 

**Description:**

Order Status Search provides real-time visibility into HP order statuses, enabling users to quickly search and track orders by key attributes. It helps monitor fulfillment progress, identify delays, and support faster decision-making across sales and operations. 
 
**Responsibilities:**

- Developed Power BI dashboards with advanced DAX (CALCULATE, SUMX, FILTER, IF, SWITCH, DIVIDE, RANKX, DATEADD), implemented Row-Level Security (RLS) and scheduled data refreshes, and integrated incremental Azure Fabric pipelines to ensure secure, reliable, and 
  high-performance reporting.
- Built and optimized data models using multiple data sources to ensure accurate, consistent, and scalable reporting across sales and finance teams. 
- Collaborated with sales, finance, and leadership stakeholders to translate business requirements into actionable visual insights. 
- Reduced report refresh time by 35% through DAX and model optimization, relationships, and data transformations.
