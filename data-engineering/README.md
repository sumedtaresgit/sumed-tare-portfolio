**Project 2:** KYC Maintenance & Federal Data Lake (FDL) Migration.

**Role:** Data Quality & KYC Operations Analyst.

**Technologies:** Power BI, SSMS, Databricks, DAX Studio, Power App, ThoughtSpot, Postgres, VS Code, Github

**Description:**

Know Your Customer (KYC) objective is to offer insights; a 360 view of the end customer Install Base and their buying patterns to increase the supply sales and the HW renewing or sales.

**Responsibilities:**

**Operations & Support**

- Performed weekly and monthly data refresh activities using Dataiku, including loading and refreshing data from multiple source systems.
- Monitored Dataiku pipelines to ensure successful data ingestion and processing.
- Investigated and resolved legacy and source-level data issues impacting downstream reporting systems 

**Responsibilities:**

**Migration (FDL / Databricks)**

- Analysed and implemented Dataiku (Data Science Studio) tables into the Federal Data Lake using Databricks.
- Converted PySpark recipes to SQL and raised pull requests using GitHub/VS Code.
- Prepared and shared technical and validation documentation for each migrated object.
- Performed basic data structure, schema, and data quality checks prior to migration approval.
- Designed and developed Power BI reports and dashboards, building optimized semantic models with relationships, hierarchies, & DAX measures to support accurate & performant analytics.
- Implemented Row-Level Security (RLS), scheduled refreshes, and performance tuning in Power BI to ensure secure, reliable, and scalable reporting.
- Created and maintained ThoughtSpot data models (Worksheets) to enable self-service 
analytics for business users. Developed ThoughtSpot Liveaboards with interactive 
visualizations and KPIs, supporting real-time insights for sales and business stakeholders.
